# Data Cleaning: Hotel Booking Demand Dataset

## Objective
- Handle missing values
- Detect and treat outliers
- Provide explanations for each step

## Dataset
Hotel Booking Demand (Kaggle)

## Steps Performed
- Identified hidden missing values using code
- Handled missing values based on feature type
- Detected outliers using statistical summary and boxplots
- Treated outliers using the IQR method

## Files
- `Data_Cleaning_Hotel_Booking.ipynb` – Jupyter Notebook with code and explanations
- `hotel_bookings.csv` – Dataset used

## Outcome
The dataset is cleaned and ready for further analysis or machine learning.
